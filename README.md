# Cloud Cost Lens - Cloud Cost Templates

This repository contains the official templates for **Cloud Cost Lens** ([cloudcostapi.com](https://cloudcostapi.com)).

## About

Cloud Cost Lens is a cloud cost management platform that helps teams monitor, analyze, and optimize their cloud spending. This repository holds the provider integration templates that are made **publicly available** to promote transparency and trust with our users.

## Templates

Templates are organized by cloud provider under the `templates/` directory.

```
templates/
└── aws/   # Amazon Web Services templates
```

### AWS

The `templates/aws/` directory contains templates for connecting to Amazon Web Services (AWS). These templates are used to grant Cloud Cost Lens the necessary read-only permissions to retrieve cost and usage data from your AWS account.

## Public Availability

These templates are made publicly available so that customers and prospective users can review exactly what permissions and configurations are required before deploying anything into their environment. This openness is a core part of the Cloud Cost Lens commitment to trust and security.
